# Política de Seguridad

Nos tomamos en serio la seguridad de nuestros proyectos. Si descubre alguna vulnerabilidad de seguridad, le pedimos que la informe de manera responsable.

## Versiones Soportadas

Actualmente, solo las versiones principales y menores más recientes de este proyecto están siendo mantenidas con actualizaciones de seguridad.

## Cómo Reportar una Vulnerabilidad

Si cree que ha encontrado una vulnerabilidad de seguridad en este proyecto, **no la revele públicamente**. Por favor, informe el problema de forma privada utilizando la función de informes de vulnerabilidad privada de GitHub o enviando un correo electrónico al equipo de seguridad (actualice con la dirección de correo electrónico real).

Incluya tanta información como sea posible para ayudarnos a comprender y reproducir el problema:

- Una descripción detallada de la vulnerabilidad.
- Los pasos para reproducirla.
- El impacto potencial.

Revisaremos su informe lo antes posible y trabajaremos con usted para solucionar el problema. Agradecemos su ayuda para mantener seguros nuestros proyectos.
